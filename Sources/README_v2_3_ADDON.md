# Virtual Internet Engine — rychlý start

Otevři Terminal ve virtuálním MacBooku a spusť:

```bash
internet help
app search vpn
download app lab-vpn
install app lab-vpn
run app lab-vpn
vpn connect omnicore
```

Vyhledávání přes Safari/CLI:

```bash
safari search vpn
google search blue team backups
internet search cloudflare dns
```

Tor simulátor:

```bash
app search tor
download app tor-browser
install app tor-browser
run app tor-browser
tor status
tor search privacy
```

Cloudflare/cloudflared simulátor:

```bash
app search cloudflare
download app cloudflared
install app cloudflared
cloudflare dns corp.local
cloudflare trace lab-web.corp.local
```

Homebrew zůstává dostupné:

```bash
brew search network
brew install nmap
nmap -sV -Pn 10.10.10.20
```
